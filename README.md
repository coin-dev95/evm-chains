# EVM-based Chains

Listed by chainId according to EIP-155

Data source available on `_data/chains.json`

Example Format

```json
{
  "name": "Ethereum Mainnet",
  "short_name": "eth",
  "chain": "ETH",
  "network": "mainnet",
  "chain_id": 1,
  "network_id": 1,
  "rpc": ["https://mainnet.infura.io/"]
}
```
